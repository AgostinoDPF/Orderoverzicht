⚙ GitHub opslag instellen
Hiermee worden je geïmporteerde Excel-bestanden en ETA-updates automatisch opgeslagen in een GitHub-repo.
Werkt op elk apparaat, elke browser — data verdwijnt nooit meer.
Snelle setup (2 min):
1. Ga naar github.com/new → maak een private repo (bijv. dp-logistics-data)
2. Ga naar github.com/settings/tokens/new → genereer een token met repo scope
3. Vul hieronder in en klik Opslaan
De repo moet al bestaan. Private is aanbevolen.
Wordt alleen lokaal opgeslagen in localStorage. Nooit naar derden verstuurd.
